# Django Todo App with User Authentication

A simple Todo application built with Django that allows users to register, log in, and manage their daily tasks.

## Features

* User Registration
* User Login / Logout
* Add new tasks
* Set task priority (Low / Medium / High)
* Set due date
* Mark tasks as completed

## Tech Stack

* Python
* Django
* SQLite
* HTML
* CSS

## Installation

1. Clone the repository

```
git clone https://github.com/demon1232/Django-Todo-App.git
```

2. Navigate to project folder

```
cd Django-Todo-App
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run migrations

```
python manage.py migrate
```

5. Start the development server

```
python manage.py runserver
```

Then open:
http://127.0.0.1:8000/
